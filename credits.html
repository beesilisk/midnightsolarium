here are the resources i used while building this website. maybe they'll be useful for you as well?

Layout:

repth.neocities.org/theme - i used Theme 12 as a template

Fonts:

TheGoodMonolith - the font used in the headers

Roboto Mono - pretty much everything else

Icons:

Font Awesome - has lots of icons (some free, some paid) such as the ones in the nav bar

Multi-Status Feed:

Bechno Kid - as seen on 'about webmaster' page; pretty much followed her tutorial to the letter 'cause i havent learned js (￢_￢;)

Misc:

Dither Machine - name-your-own-price software that I use to make dithered gradients (like the one in the background)

ai.robots.txt - Cory Dransfeldt's list of AI web crawlers. this is the base for my robots.txt file

Code Editors:

Notepad++ - the text editor where I do the bulk of my frontend coding (that might have to change once I finally switch to Linux)

CodePen - an online HTML/CSS/JavaScript IDE with a real-time preview; it's handy for experimenting and making small adjustments
